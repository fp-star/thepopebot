## Key Features (Current State)

### Core Capabilities
1. **Autonomous AI Agents** - Self-modifying agents via pull requests
2. **Security-First Design** - API keys filtered at process level
3. **Free Compute** - Uses GitHub Actions (2,000 free minutes/month on free plan)
4. **Git-Based History** - Every action is a commit
5. **Telegram Chat Interface** - Conversational interface for job creation
6. **Cron Jobs** - Scheduled automated tasks
7. **Webhook Triggers** - Event-driven automation
8. **Docker-Based Execution** - Containerized agent runtime

### Architecture
- **Two-Layer Design:**
  1. Event Handler (Node.js/Express) - Orchestration, webhooks, Telegram, cron
  2. Docker Agent - Pi coding agent execution in isolated containers

### Tech Stack
- Node.js 22
- Pi coding agent (@mariozechner/pi-coding-agent)
- Playwright + Chromium (browser automation)
- Docker containers
- GitHub Actions
- Anthropic Claude (primary LLM)
- Telegram Bot API

## Getting Started

To use thepopebot:

1. **Fork** the repository
2. **Enable GitHub Actions** in your fork
3. **Run setup wizard:** `npm run setup`
4. **Message your Telegram bot** to create jobs

### Prerequisites:
- Node.js 18+
- Git
- GitHub CLI
- ngrok (for local development)
- Anthropic API key
